# Retail Sales Performance Dashboard

## 📌 Project Overview

This project is an interactive Retail Sales Performance Dashboard built in Microsoft Excel to analyze sales performance across different product categories, customer segments, and states.

The dashboard enables users to filter sales data dynamically using slicers and visualize key business metrics for better decision-making.

---

## 🎯 Business Problem

Retail businesses need an efficient way to monitor sales performance across different regions, customer segments, and product categories. This dashboard provides an interactive solution for exploring sales trends and identifying performance patterns.

---

## 🛠️ Tools Used

- Microsoft Excel
- Power Query
- Power Pivot
- Pivot Tables
- Pivot Charts
- Slicers
- Filled Map Chart

---

## 📊 Dashboard Features

- Product Category slicer
- State slicer
- Customer Segment slicer
- Sales by Product Category
- Sales Distribution by State
- Sales by Customer Segment
- Maximum Sales KPI
- Customer KPI
- Product Count KPI
